这里说明的是各个返回码对应的情况

400 : 用户注册成功
401 : 用户注册失败
402 : 登录成功
403 : 用户名或密码格式错误
404 : 密码错误
405 : 不存在该用户
406 : 邮箱格验证码发送失败
407 : 邮箱验证码发送成功
408 : 邮箱已经注册过了
409 : 邮箱格式错误
410 : 找到相关文章
411 : 未找到相关文章
412 : 找到文章序号对应的标题
413 : 未找到文章序号对应的标题
414 : 成功返回文章内容
415 : 文章内容返回失败
416 : 文章上传成功
417 : 文章上传失败
418 : 头像上传成功
419 : 头像上传失败
420 : 获取文章成功
421 : 获取文章失败
422 : 获取评论成功
423 : 获取评论成功
424 : 发送评论成功
425 : 发送评论失败
426 : 点赞成功
427 : 收藏成功
